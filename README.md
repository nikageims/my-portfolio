# 🚀 NIKA GEIMS | Portfolio

კეთილი იყოს თქვენი მობრძანება ჩემს პერსონალურ პორტფოლიო ვებსაიტზე. ეს პროექტი წარმოადგენს ჩემს გამოცდილებას ვებ-დეველოპმენტში, დიზაინსა და კონტენტის შექმნაში.

🌐 **Live Demo:** [my-portfolio-ten-bice-28.vercel.app](https://my-portfolio-ten-bice-28.vercel.app/)

## 🛠️ ტექნოლოგიური სტეკი

პროექტი აწყობილია თანამედროვე ხელსაწყოების გამოყენებით:

* **Framework:** React.js
* **Styling:** Tailwind CSS (Cyberpunk/Modern UI დიზაინით)
* **Icons:** Lucide React / FontAwesome
* **Deployment:** Vercel

## ✨ ფუნქციები

- **Modern & Cyberpunk UI:** მუქი თემა ნეონის ელემენტებით.
- **Responsive Design:** სრულად ადაპტირებული ყველა ზომის ეკრანზე.
- **Project Showcase:** ჩემი ნამუშევრებისა და პროექტების (მაგ: CyberX) დეტალური აღწერა.
- **Skills Section:** ჩემი ტექნიკური უნარების ვიზუალური პრეზენტაცია.
